Cyber Link Lock

Cyber Link Lock is an open-source tool for adding password protection to links by encrypting and decrypting URLs. When a user opens a protected link, they must enter the correct password to access the original website.

All encrypted data is stored directly inside the generated link. No information is stored on servers, and there are no cookies, tracking, or user accounts. This ensures complete privacy and user control.

This project is developed as a Final Year Project by Ganesh Kumar.
